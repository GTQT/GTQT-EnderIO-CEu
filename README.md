# GTQT EnderIO CEu

本仓库用于对 Minecraft 1.12.2 GTQT环境下的特供版本。
该mod会追加一些GTQT特有的物品和内容 不保证其他环境的运行稳定性
EnderIO 最初由 EnderIO Team 开发。目前EnderIO CEu由Dimensional Team维护。
请不要因 EnderIO CEu GTQT特供版本的问题联系原团队；本项目与他们没有从属关系。
